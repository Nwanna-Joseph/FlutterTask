## Getting Started

### Build APK:
```
flutter build apk --release --target=lib\presentation\screen\main.dart
```

### Run Debug
```
flutter run --target=lib\presentation\screen\main.dart
```

### Run Test
```
flutter test test/task_repository_tests/mock_flutter_secure_storage.dart
```
